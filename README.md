# strukt-programavimas

Struktūra: 
1. Pasirenkamos žmonių klasės, kurios duoda smulkius bonusus arba galima rinktis be papildomų pastiprinimų(sunkesnis rėžimas)
2. Kiekvieną dieną galima: rinkti maistą(+ arba - gyvybės/energija), rinkti resursus(rinkti metalo gabalus iš kurių galima gaminti šarvus/ginklus kurie padės kovoti prieš monstrus), arba ilsėtis(+ energija arba esi užpultas).
3. Iš šių veiksmų galima atlikti tik po kartą ir tada reikia praleisti dieną naujiems veiksmams.
4. Atliekant šiuos veiksmus gali būti užpultas random sugeneruoto monstro ir priklausomai nuo pasaulio(WORLD) ir tavo ginklų bus pritaikyta monstro gyvybės/ataka.
5. Tikslas kuo ilgiau išgyventi
6. Galima save/load progresą, kas tuo pačiu metu sugeneruos html+CSS failą su highscore lentele
